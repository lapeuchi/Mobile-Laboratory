2. 팀 소개
모바일 실험실은 AR, 이미지 트래킹 기술을 활용한 교과서 컨텐츠 체험 플랫폼 입니다.

학교에서 과학 수업을 하면 도중 안전상의 문제, 비용 문제 등으로 실험 수업을 하지 못하는 경우가 대다수 입니다.
이를 해결하기 위해 AR 기술을 활용하여 교과서 컨텐츠를 개발하게 되었습니다.
개발 목표
1. AR 인식 AI에게 대량의 교과서 이미지를 학습시켜 높은 인식률 추구
2. 교과서 내 탐구 컨텐츠들에 해당하는 컨텐츠를 제작

3. 사용 기술
AR SDK에서 우위를 점하고 있는 MAXST는 정확한 오브젝트 인식률을 자랑합니다.
모바일 실험실은 MAXST ImageTracker를 Unity와 융합하여 사용합니다.

6. 프로젝트 추진 결과
모바일 실험실의 상황은 다음과 같습니다.
1. 교과서 선택: 인식 범위를 줄여 인식률 증가
2. 이미지 인식: 인식 후 진행되는 탐구형 컨텐츠, 알아보기 컨텐츠
3. 알아보기 컨텐츠: 진행 순서가 없는 관찰식 컨텐츠
4. 탐구형 컨텐츠: 진행 순서가 있는 실험 컨텐츠
6. 인식 코드 최적화: MAXST SDK를 모바일 실험실에 맞게 수정하여 기기의 과부하 개선
모바일 실험실의 추후 목표
다양한 교과서의 컨텐츠들을 개발하여 많은 학생들이 사용할 수 있도록 할 것입니다.
5. 정리하기: 탐구형 컨텐츠 이후 교과서의 중요 내용을 정리한 팝업 출력
